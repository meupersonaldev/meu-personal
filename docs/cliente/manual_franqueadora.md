# Manual da Franqueadora - Meu Personal

Este documento descreve as funcionalidades disponíveis para o perfil **Franqueadora** no sistema Meu Personal. O objetivo deste perfil é gerenciar a rede de franquias, definir políticas globais e acompanhar o desempenho geral.

## Visão Geral

A Franqueadora é o nível mais alto de administração do sistema. Ela tem visão total sobre todas as unidades (franquias) e usuários.

### 4. Políticas e Configurações
Definição de regras que se aplicam a toda a rede:
- **Pacotes de Créditos:** Criação e edição dos pacotes de aulas que serão vendidos pelos alunos.
- **Pacotes de Horas:** Definição dos pacotes de horas que os professores podem adquirir (se o modelo de negócio exigir).
- **Regras de Negócio:** Configurações globais de cancelamento, prazos e valores base.
